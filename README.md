# InnoCrate Website

Official website for **InnoCrate**, a product engineering and technology studio focused on building reliable, scalable, and purpose-driven software.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- EmailJS

## Setup & Installation

### Prerequisites
- Node.js (v18 or later)
- npm

### Install & Run

```bash
git clone <YOUR_GIT_REPO_URL>
cd <PROJECT_NAME>
npm install
npm run dev

npm run build
npm run preview
